LocalLink UG – Job Board Platform

Overview
LocalLink UG is a web-based job board designed to connect job seekers with employers in Uganda, helping reduce unemployment by improving access to opportunities.

Features
- User registration and login
- Job posting and management (employers)
- Job application system (job seekers)
- Admin dashboard
- CV upload functionality
- Application tracking system

Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- Tailwind CSS

 Challenges & Solutions
- Handling user authentication securely → implemented password hashing
- Managing job applications → designed relational database structure
